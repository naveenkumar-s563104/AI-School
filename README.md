# Astro AI School Full Website

This is a clean Astro + Vercel-ready AI School website.

## Pages included

- Home: `/`
- About: `/about`
- Programs: `/programs`
- AI Lab: `/ai-lab`
- Online School: `/online-school`
- Admissions: `/admissions`
- Student Life: `/student-life`
- News: `/news`
- Contact: `/contact`

## Local development

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:4321
```

## Build

```bash
npm run build
```

## Vercel settings

- Framework: Astro
- Build command: `npm run build`
- Output directory: `dist`
